# 惨剧轮回 (Tragedy Looper) 游戏规则摘要

## 1. 游戏概述

### 1.1 基本信息
- **游戏类型**：非对称对抗推理
- **玩家数量**：2-4 人（1 名剧作家 vs 1-3 名主人公）
- **游戏时长**：30 分钟 - 4 小时（视剧本而定）

### 1.2 角色与胜利条件

| 阵营 | 玩家 | 胜利条件 |
|--------|--------|-----------|
| 剧作家 | 1 人 | 在所有轮回成功引发惨剧，且不被主人公猜中所有角色身份 |
| 主人公 | 1-3 人 | 避免惨剧发生，或正确推理所有角色身份（最终决战） |

## 2. 核心概念

### 2.1 轮回 (Loop)
- 游戏在固定次数的轮回中进行
- 每次轮回结束后，角色状态重置
- 主人公在轮回中获得的信息会被保留用于推理

### 2.2 天数 (Day)
- 每个轮回由数天组成
- 每天包含：黎明 → 行动阶段 → 夜晚

### 2.3 指示物系统

#### 友好 (Goodwill)
- **用途**：使用角色能力
- **来源**：主人公行动牌（友好+1/+2）、角色能力
- **限制**：剧作家可用"禁止友好"阻止增加

#### 不安 (Anxiety)
- **用途**：触发事件
- **来源**：行动牌（不安+1）、传谣人能力
- **限制**：主人公可用"不安-1"减少

#### 密谋 (Intrigue)
- **用途**：主线失败条件、医院事故、杀手能力
- **来源**：行动牌（密谋+1/+2）、主谋能力
- **限制**：主人公无法减去

## 3. 游戏流程

### 3.1 游戏开始
1. 剧作家选择剧本，向主人公展示**公开信息表**
2. 配置版图，决定首轮队长
3. 配置角色，移除指示物，分发行动牌

### 3.2 一天的流程

| 阶段 | 内容 |
|--------|------|
| **3.1 黎明** | "亲友"角色友好 +1 |
| **3.2 剧作家创作阶段** | 剧作家暗置 3 张行动牌 |
| **3.3 主人公行动阶段** | 主人公依次暗置 3 张行动牌（每人每张） |
| **3.4 行动牌结算** | 1. 移动牌优先结算<br>2. 邪教徒能力<br>3. 指示物牌结算 |
| **3.5 剧作家能力阶段** | 主谋 & 传谣人 & 医生能力（无视友好+友好≥2） |
| **3.6 主人公能力阶段** | 友好达标的角色发动能力 |
| **3.7 事件阶段** | 检查是否有事件发生并处理 |
| **3.8 夜晚** | 杀人狂 & 杀手能力 |

### 3.3 行动牌类型

> 详细卡牌列表请参考 [行动牌系统](./action-cards.md)

行动牌分为四大类：

| 类型 | 剧作家 | 主人公 |
|------|--------|--------|
| 移动 | 斜向移动*, 禁止移动*, 横向, 纵向 | 禁止移动*, 横向, 纵向 |
| 友好 | 禁止友好 | 友好+1, 友好+2* |
| 不安 | 不安+1(×2), 禁止不安 | 不安+1, 不安-1* |
| 密谋 | 密谋+1, 密谋+2* | 禁止密谋 |

> * = 每轮限一次

**放置规则：**
- 每个角色上最多暗置一张主人公行动牌
- 放置的行动牌需要暗置，主人公之间也不知道互相放置了什么牌
- 这可能会有出乎意料的精彩配合，也可能导致糟糕的结果

## 4. 事件系统

### 4.1 事件发生条件
事件发生必须同时满足以下三个条件：

```
I.  今天有事件（公开信息表指定）
II. 该事件的当事人仍存活
III. 当事人的不安 ≥ 不安限度
```

### 4.2 事件处理
- 声明事件发生并描述现象
- 角色死亡则轮回立即结束
- 医院事故发生但医院无密谋 → 声明事件已发生但无现象

## 5. 版图与角色

### 5.1 版图区域（固定位置）
```
┌─────────┬─────────┐
│ 医院   │   神社   │
│(左上) │  (右上)  │
├─────────┼─────────┤
│ 都市   │   学校   │
│(左下) │  (右下)  │
└─────────┴─────────┘
```

### 5.2 角色卡片信息
- **角色名**
- **不安限度**：触发事件所需不安值
- **初始区域**：游戏开始位置
- **禁行区域**：角色绝对无法进入的区域
- **角色特性**：特殊标签（学生、少女等）
- **友好能力**：友好指示物达到要求时可发动的效果

## 6. 角色身份

### 6.1 常见身份列表

| 身份 | 特性 | 能力说明 |
|------|--------|----------|
| 杀手 | 无视友好 | 与某人同区域且密谋≥4 时可杀死该角色 |
| 杀人狂 | - | 夜晚与某人同区域时，杀人狂死亡并杀死该角色 |
| 主谋 | 无视友好 | 与角色同区域时可给予密谋+1 |
| 传谣人 | - | 给予同区域角色不安+1 |
| 邪教徒 | 必定无视友好 | 需拒绝友好能力发动 |

### 6.2 FS-01 模组角色
- **学生**（男/女）
- **巫女**
- **医生**
- **异界人**（禁入医院）

## 7. 剧本系统

### 7.1 剧本构成
每个剧本包含：
1. **主线 (Rule Y)**：核心失败条件
2. **支线 (Rule X)**：额外规则，增强或改变主线
3. **角色**：登场的 NPC 及其身份配置
4. **事件**：每天的惨剧安排

### 7.2 FS-01 新手模组
- 主人公**不能通过推理身份获胜**
- 目标：避免惨剧发生

## 8. 特殊规则

### 8.1 能力结算顺序
1. **强制能力**优先任意能力同时结算
2. 多个强制能力**同时进行**
3. 多个任意能力由**剧作家决定顺序**

### 8.2 同一区域规则
- "同一区域"包含该角色自身
- 移动牌可改变区域从而影响能力发动

### 8.3 友好能力使用
- 能力结算后**不移除**友好指示物
- 每轮回限一次的能力在轮回结束后重置
- 无限制能力可在每天发动

## 9. 胜利与失败

### 9.1 主人公胜利
- 所有轮回结束时，未触发惨剧
- 或最终决战时正确推理所有角色身份

### 9.2 剧作家胜利
- 所有轮回中，至少一次成功引发惨剧
- 且主人公未猜中所有角色身份

### 9.3 轮回立即结束
- **关键人物**死亡
- **主人公**死亡
- 事件导致角色死亡

---

**注**：本规则为简化版，用于指导 Web 游戏开发。具体细则请参考官方规则书。
