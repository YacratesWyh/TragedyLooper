# 行动牌系统

## 概述

行动牌分为四大类：**移动**、**友好**、**不安**、**密谋**

- 🔴 红色边框 = 剧作家行动牌
- 🔵 蓝色边框 = 主人公行动牌  
- ⭐ 带 * 标记 = 每轮限一次

## 移动牌叠加规则

当同一角色身上有多张移动牌时，使用以下叠加规则计算最终移动方向：

| 方向1 | 方向2 | 结果 |
|-------|-------|------|
| 横向 | 横向 | 横向 |
| 竖向 | 竖向 | 竖向 |
| 斜向 | 斜向 | 斜向 |
| 横向 | 竖向 | 斜向 |
| 斜向 | 横向 | 竖向 |
| 斜向 | 竖向 | 横向 |

**规则总结：**
- 相同方向 = 相同方向
- 横+竖 = 斜向
- 斜+方向 = 另一个方向（斜+横=竖，斜+竖=横）

**禁止移动牌**会完全取消该角色的所有移动。

## 卡牌列表

### 移动牌

| 卡牌 | 剧作家 | 主人公 | 效果 |
|------|:------:|:------:|------|
| 斜向移动* | ✅ | ❌ | 角色斜向移动一格 |
| 禁止移动* | ✅ | ✅ | 禁止该角色本回合所有移动 |
| 横向移动 | ✅ | ✅ | 角色横向移动一格 |
| 纵向移动 | ✅ | ✅ | 角色纵向移动一格 |

### 友好牌

| 卡牌 | 剧作家 | 主人公 | 效果 |
|------|:------:|:------:|------|
| 禁止友好 | ✅ | ❌ | 抵消该角色上对方的友好牌 |
| 友好+1 | ❌ | ✅ | 角色友好+1 |
| 友好+2* | ❌ | ✅ | 角色友好+2 |

### 不安牌

| 卡牌 | 剧作家 | 主人公 | 效果 |
|------|:------:|:------:|------|
| 禁止不安 | ✅ | ❌ | 抵消该角色上对方的不安牌 |
| 不安+1 | ✅(×2) | ✅ | 角色不安+1 |
| 不安-1* | ❌ | ✅ | 角色不安-1 |

> ※ 剧作家有**两张**不安+1

### 密谋牌

| 卡牌 | 剧作家 | 主人公 | 效果 |
|------|:------:|:------:|------|
| 密谋+1 | ✅ | ❌ | 角色/地点密谋+1 |
| 密谋+2* | ✅ | ❌ | 角色/地点密谋+2 |
| 禁止密谋 | ❌ | ✅ | 抵消该目标上对方的密谋牌 |

> ※ 密谋牌可以放在**角色**或**地点**上

## 牌组总览

### 剧作家牌组（10张）

```
移动: 斜向移动*, 禁止移动*, 横向移动, 纵向移动
友好: 禁止友好
不安: 不安+1, 不安+1, 禁止不安
密谋: 密谋+1, 密谋+2*
```

### 主人公牌组（8张）

```
移动: 禁止移动*, 横向移动, 纵向移动
友好: 友好+1, 友好+2*
不安: 不安+1, 不安-1*
密谋: 禁止密谋
```

## 放置规则

1. **每天每人放置3张牌**
2. **每个角色上最多放一张自己的牌**（同一角色可以有剧作家1张+主人公1张）
3. **放置的牌需要暗置**（面朝下）
4. **主人公之间也不知道互相放置了什么牌**

## 多主人公规则

- 游戏支持 **1-3 名主人公玩家**
- **每个主人公玩家各有一套完整牌组**
- 因此"每轮限一次"的牌，主人公方**最多可以打出3次**（每人各用一次）
  - 例：3个主人公 = 3张友好+2*、3张不安-1*、3张禁止移动*

## 结算规则

1. **移动牌先结算**
   - 如果同一角色上有"禁止移动"，所有移动牌无效
   
2. **指示物牌后结算**
   - 如果有"禁止XX"牌，对方的同类型牌被抵消
   
3. **地点上的牌**
   - 只有**密谋牌**放在地点上才有效果
   - 其他牌放在地点上功能无效（但可作为欺骗战术）

## 战术提示

- 放"假牌"到地点上欺骗对手（对方只能看到有牌，不知道内容）
- 主人公之间的牌也是互相保密的，可能产生精彩配合，也可能导致糟糕结果
- "禁止"牌是防守利器，用来保护关键角色或破坏对手计划

## 阻止事件的方法

事件触发需要同时满足：当事人存活 + 不安 ≥ 限度 + （某些事件）地点条件

| 方法 | 适用事件 | 说明 |
|------|----------|------|
| 降低不安 | **所有事件** | 使当事人不安 < 不安限度 |
| 移动当事人 | **特定事件** | 如医院事故需当事人在医院，移走可阻止 |
| 禁止不安 | **所有事件** | 抵消剧作家的不安+1 |

> ⚠️ 移动当事人只对**有地点要求**的事件有效，普通谋杀事件不受地点影响
