# 游戏资产说明

## 角色立绘

### characters-grid.png

**说明**：请将规则书中的角色立绘图片保存为此文件名

**要求**：
- 文件名：`characters-grid.png`
- 尺寸：3840 x 2160 像素
- 格式：8列 x 4行网格
- 每个角色：480 x 540 像素

**当前状态**：
- ⚠️ 需要手动将角色立绘图片保存到此目录
- 📁 路径：`public/assets/characters-grid.png`

### 网格布局

```
[男学生] [女学生] [巫女] [女学生2] [医生] [上班族] [情报员] [病患]

[科学家] [宠物] [小女孩] [双子] [占卜师] [幻影] [妖狐] [巫女2]
```

## 使用方式

1. 将角色立绘图片保存为 `public/assets/characters-grid.png`
2. 确保尺寸正确：3840 x 2160 像素
3. 系统会自动通过 CSS background-position 裁剪显示

## 备用方案

如果没有图片资源，系统会显示：
- 纯色背景 + 角色名称文字
